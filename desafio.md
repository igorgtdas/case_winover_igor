Desafio técnico — Engenheiro de IA Conversacional

Desenvolva uma solução mínima funcional de IA conversacional para um cenário interno de suporte e operação da empresa fictícia AtlasShop.

A solução deve ser capaz de usar a base fornecida para:

responder perguntas a partir de documentação interna
consultar dados operacionais estruturados
sustentar um fluxo conversacional coerente

Você terá acesso a:

documentos em knowledge/
dados estruturados em data/
contexto adicional em cenario.md

A abordagem é livre. Escolha a arquitetura, stack, modelo e forma de orquestração que fizerem mais sentido para o problema.

A entrega deve incluir:

código-fonte
README com instruções de execução
breve documento com decisões técnicas e trade-offs
diagrama simples da arquitetura e do fluxo principal

Não é necessário construir interface sofisticada.
Simplificações são aceitáveis, desde que fiquem claras.
